Query 1: What are all the scores for the selected country?

User inputs a country. Receives their HDI rank, FSI rank, and all indicators.


Query 2: What countries have a strong score in the selected indicator?

User selects an indicator from a dropdown list. We get all the countries that have an especially good score (lower is good, so 3 or lower?) in that indicator

Query 3: What countries have a weak score in the selected indicator?

User selects an indicator from a dropdown list. We get all the countries that have an especially BAD score (higher is bad, so 7 or higher?) in that indicator
